# Student Performance Analytics System

## Introduction

Iss project purpose student ke records ko manage karna aur unke marks ka analysis karna hai.

Is project me students ko add, update, delete aur display kar sakte hain. Iske alawa statistics, probability aur different charts ka use karke student performance ko analyze kiya gaya hai.

---

## Features

### Student Record Management

* Add New Student
* Update Student Marks
* Delete Student Record
* Display All Students

### Statistics

* Mean
* Median
* Mode
* Range
* Variance
* Standard Deviation
* Quartile 1 (Q1)
* Quartile 3 (Q3)
* Inter Quartile Range (IQR)

### Probability

* Probability of Student having Marks >= 80
* Conditional Probability P(Marks >= 90 | Marks >= 80)

### Charts

Project me following charts generate kiye gaye hai:

* Histogram
* Bar Chart
* Box Plot
* Scatter Plot
* Line Chart

Saare charts **screenshots folder** me save hote hai.
